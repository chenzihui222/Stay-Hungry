# 科创金融研究

> **Tech Finance Hub** - 科创板数据 | VC市场分析 | 政策解读

一个专注于科技创新与金融投资领域的研究网站，参考 [VC Radar](https://github.com/chenzihui222/News-about-VC) 的深色科技风格设计。

## 🚀 在线预览

**GitHub Pages**: https://chenzihui222.github.io/Stay-Hungry

## 📊 功能模块

### 1. 科创板数据 (STAR Market Data)
- 科创板实时行情与统计数据
- 上市公司概览与市场表现
- 行业分布与热门赛道分析
- 涨跌排行与成交数据

### 2. VC市场分析 (VC Market Analysis)
- 风险投资市场趋势洞察
- 热门投资赛道与交易数据
- 活跃投资机构与近期大额交易
- 季度/年度投资报告

### 3. 政策解读 (Policy Insight)
- 监管政策与市场动态追踪
- 科创金融相关政策解读
- 政策影响分析与行业指引
- 法规更新与合规提示

## 🛠 技术栈

- **框架**: Next.js 14 + React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **部署**: GitHub Pages

## 📦 安装与运行

```bash
# 克隆项目
git clone https://github.com/chenzihui222/Stay-Hungry.git
cd Stay-Hungry

# 安装依赖
npm install

# 开发模式
npm run dev

# 构建静态网站
npm run build

# 部署到GitHub Pages
npm run deploy
```

## 🎨 设计风格

- **主题**: 深色科技风 (Dark Tech)
- **主色调**: 青蓝色 (#00d4ff) + 深空黑 (#0a0a0f)
- **字体**: Inter (正文) + JetBrains Mono (数字)
- **布局**: 响应式设计，支持移动端和桌面端

目前网站使用的是示例数据，后续将根据您提供的数据源进行API对接：

- **VC市场数据**: IT桔子、投中网、Crunchbase等
